# Platform Classification Update (Based on Gemini Research)

## 📊 New Platform Classification Structure

Based on comprehensive analysis and feedback from Gemini AI research, we've reorganized the platform catalog into 4 main categories instead of difficulty levels.

---

## 🛠️ Category 1: For Experienced Miners (HiveOS/Old Rigs)

**Perfect for:** People who have mining rigs, HiveOS experience, or simply older GPU equipment

### Clore.ai
- **Type:** AI Computing / Pay-per-task
- **Earning Model:** Complete task-based income
- **Min GPU:** Any
- **Profit Level:** Low-Medium
- **Setup:** Medium complexity
- **Legacy Support:** ✅ Yes (supports old GPUs)
- **Scalability:** Medium
- **Commission:** 15-25%
- **Best For:** Miners converting to AI tasks

### OctaSpace
- **Type:** Decentralized Computing / Proof-of-Work
- **Earning Model:** PoW rewards + fees
- **Min GPU:** Any
- **Profit Level:** Medium
- **Setup:** Medium complexity
- **Legacy Support:** ✅ Yes (crucial for this category)
- **Scalability:** Yes
- **Commission:** 2-5% (very competitive)
- **Best For:** Those wanting to exit centralized mining

---

## 💻 Category 2: For Single Powerful PC Owners

**Perfect for:** Gamers/regular users with a good PC who want easy passive income

### Salad
- **Type:** P2P Network / Passive Income
- **Earning Model:** Run and forget - passive per hour
- **Min GPU:** Gaming GPU
- **Profit Level:** Low
- **Setup:** **Very Easy** (1-click)
- **Legacy Support:** ✅ Yes
- **Scalability:** No (1 PC per account)
- **Commission:** 25%
- **Best For:** Absolute beginners, no tech knowledge needed

### GamerHash
- **Type:** Distributed Computing
- **Earning Model:** Per-task + passive hybrid
- **Min GPU:** Any
- **Profit Level:** Low-Medium
- **Setup:** Easy
- **Legacy Support:** ✅ Yes
- **Scalability:** Medium
- **Commission:** 20%
- **Best For:** Those wanting flexibility between passive and active

### Nosana
- **Type:** Decentralized Network / Crypto
- **Earning Model:** Token rewards for rendering jobs
- **Min GPU:** RTX 3080+
- **Profit Level:** Medium
- **Setup:** Medium
- **Legacy Support:** ❌ No (requires modern GPU)
- **Scalability:** Yes
- **Commission:** 5-10%
- **Best For:** Crypto-interested users with modern GPUs
- **Docs:** https://docs.nosana.io
- **Key Difference:** Decentralized, growing ecosystem

---

## 💼 Category 3: For Serious Business Builders

**Perfect for:** Those investing in GPUs for actual business/ROI calculations

### Vast.ai
- **Type:** Peer-to-Peer GPU Rental Marketplace
- **Earning Model:** Hourly rental rates (most predictable)
- **Min GPU:** RTX 4090 / H100
- **Profit Level:** High ($2-3/hour typically)
- **Setup:** Medium
- **Legacy Support:** ❌ No
- **Scalability:** Unlimited nodes
- **Commission:** 30%
- **Best For:** Most business-focused users
- **Why:** Established, largest user base, highest reliability

### RunPod
- **Type:** Cloud Computing Platform
- **Earning Model:** Per-hour billing + provisioning bonuses
- **Min GPU:** RTX 4090 / A100
- **Profit Level:** High ($2.2/hour typical)
- **Setup:** Medium
- **Legacy Support:** ❌ No
- **Scalability:** Yes
- **Commission:** 25-30%
- **Best For:** Those who want stability + growth potential

### io.net
- **Type:** Decentralized GPU Network
- **Earning Model:** Token rewards (IO) + usage fees
- **Min GPU:** RTX 4070+
- **Profit Level:** High ($2/hour + token appreciation)
- **Setup:** Medium
- **Legacy Support:** ❌ No
- **Scalability:** Yes
- **Commission:** 5-15% (if token appreciates, can be negative!)
- **Best For:** Long-term believers in decentralization
- **Unique:** Stake IO tokens for higher rewards

### Prime Intellect
- **Type:** Premium GPU Cloud Platform
- **Earning Model:** Rental + compute share + training partnerships
- **Min GPU:** A100 / H100 (Enterprise)
- **Profit Level:** Very High ($3-5/hour typical)
- **Setup:** High (requires enterprise setup)
- **Legacy Support:** ❌ No
- **Scalability:** Limited but powerful
- **Commission:** 20%
- **Best For:** Serious businesses building production systems
- **Unique:** Direct integration with AI companies, longer contracts

### TensorDock
- **Type:** Cloud GPU Marketplace
- **Earning Model:** Hourly rental + instant payouts
- **Min GPU:** RTX 4090+
- **Profit Level:** High ($1.8-2.5/hour)
- **Setup:** Medium
- **Legacy Support:** ❌ No
- **Scalability:** Yes
- **Commission:** 25%
- **Best For:** Those wanting Vast.ai alternative with different user base
- **Unique:** Instant payouts (not daily/weekly)

---

## 🔗 Category 4: Decentralized Networks (Long-Term Believers)

### Akash Network
- **Type:** Fully Decentralized Cloud Network (Cosmos-based)
- **Earning Model:** Token rewards (AKT) for providing compute
- **Min GPU:** Any
- **Profit Level:** Medium
- **Setup:** Hard (requires node running, config)
- **Legacy Support:** ✅ Yes
- **Scalability:** Yes
- **Commission:** 2-5% (protocol fees)
- **Best For:** Those betting on blockchain infrastructure
- **Unique:** Fully on-chain, censorship-resistant
